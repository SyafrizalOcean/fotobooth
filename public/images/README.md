# 📁 Folder Images — TARUH GAMBAR NAILONG DI SINI

Taruh **4 file gambar Nailong** di folder ini dengan nama yang persis seperti berikut:

```
nailong1.png   ← muncul di pojok kiri atas frame + sebagai maskot di header
nailong2.png   ← pojok kanan atas frame
nailong3.png   ← pojok kiri bawah frame
nailong4.png   ← pojok kanan bawah frame
```

## Saran:
- Format **PNG dengan background transparan** paling bagus
- Ukuran disarankan **200x200px atau lebih** (square / 1:1 aspect ratio)
- Kalau cuma punya 1 gambar Nailong, **copy aja 4x** dengan nama berbeda (nailong1.png, nailong2.png, dst)
- Kalau file gak ada, aplikasi akan otomatis fallback ke emoji 🐲

## Mau ganti gambar?
Cukup replace file-nya, refresh browser. Done.
